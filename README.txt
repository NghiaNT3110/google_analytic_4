# google_analytic_4
- Data Source: GA4 Sample Dataset on Big Query. The dataset have a range of date from 01-11-2020 to 31-01-2021 (3 months)
- Data Storage: The data after transformed will be save as Big Query table format, expired in 60 days (Big Query SandBox/Free Tier only)
- Data Schema Details: You can check more at here: https://support.google.com/analytics/answer/7029846?hl=en&sjid=5908776957046870674-AP#zippy=%2Cdevice%2Cgeo%2Capp-info%2Ccollected-traffic-source%2Ctraffic-source%2Cstream-v%C3%A0-platform%2Cecommerce%2Citems
- Transform Data Methods: SQL - You can check SQL code in the folders
