- Purchase Table - Done: Columns in the Table: User ID, Date, URL, items name, brand, price, quantity, revenue, Transaction ID, tax, traffic name, source, medium, Session ID, Country, City, Device, Browser
- Traffic Table - Done: Columns in the Table: User ID, date, Page name, URL, traffic name, source, medium, Session ID, Country, City, Device, Browser, event name, Transaction ID, is conversion event, page view
- Time Web Table - Ongoing: Columns in the Table: = Traffic Table + Time on Page + Time on Session + Engagement Rate + Bounce Rate
